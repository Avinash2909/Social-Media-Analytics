Steps to run the steamlit app 

1)install stream lit by following command 
  "pip install streamlit"

2)Run the webapp.py in any python ide

3)open terminal or command prompt navigate to the directiory where the webapp.py is saved
  run the following command 
  (streamlit run webapp.py)

4) After running the above command streamlit will star the local server and provide you with a url
   by default it is "http://localhost:8501"

5)interact with the application in the browser
